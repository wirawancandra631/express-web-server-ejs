# Express web server with ejs template engine



Beberapa library yang di gunakan

1.Express
2.Express cookie parser
3.Express session
4.Express validator
5.Express flash
6.Ejs template engine


--- 
Tahapan memulai

```shel 
 git clone <repo url>
 ```

Mengunduh modul yang diperlukan projek
``` npm install ```
Memulai server
``` npm run serve ```
